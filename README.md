##### Installation

From your homedirectory (on Linux/MacOSX):

* `cd ~`
* `git clone git://github.com/hiasinho/hiasvim.git`
* `ln -sfn hiasvim .vim`
* `ln -sfn hiasvim/vimrc .vimrc`
* `cd .vim && make install`
